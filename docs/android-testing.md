# Android on Google Play

Overtchat is currently in closed testing for Android. Please join here:

1. Join the testers group:
   https://groups.google.com/g/overtchat-testers
2. Accept the Google Play test invite:
   https://play.google.com/apps/testing/com.overtchat.mobile
3. Install overtchat:
   https://play.google.com/store/apps/details?id=com.overtchat.mobile
4. Done
